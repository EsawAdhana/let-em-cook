# let-em-cook

Stanford CS147 final project — an "Adulting Made Easier" cooking app. The [course project page](https://web.stanford.edu/class/cs147/projects/AdultingMadeEasier/LetEmCook/) has the full design rationale, user research, and team writeup.

The app is built with Expo / React Native, uses Supabase for the backend, and pulls in Expo Google Fonts (Fredoka, Poppins). Source lives in the `let-em-cook/` subdirectory.
