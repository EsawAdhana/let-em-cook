# let-em-cook

Stanford CS147 project — an "Adulting Made Easier" cooking app.

## Overview

A React Native (Expo) app built for Stanford CS147. See the [course project page](https://web.stanford.edu/class/cs147/projects/AdultingMadeEasier/LetEmCook/) for context, design rationale, and team writeup. Source lives under the `let-em-cook/` directory.

## Stack

- Expo / React Native
- Supabase (`@supabase/supabase-js`)
- `@react-navigation/native`
- Expo Google Fonts (Fredoka, Poppins)

## Getting started

```bash
cd let-em-cook
npm install
npm start
```

Then run on a target platform:

```bash
npm run ios
npm run android
npm run web
```
